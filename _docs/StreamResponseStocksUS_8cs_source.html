104  /// <summary>
105  /// Refers to the latest relevant price of the security which is derived from multiple sources. We first look for an IEX real
106  /// time price. If an IEX real time price is older than 15 minutes, 15 minute delayed market price is used. If a 15 minute
107  /// delayed price is not available, we will use the current day close price. If a current day close price is not available,
108  /// we will use the last available closing price (listed below as previousClose) IEX real time price represents trades
109  /// on IEX only. Trades occur across over a dozen exchanges, so the last IEX price can be used to indicate the overall market price.
110  /// This will not included pre or post market prices
111  /// </summary>
112  public decimal LatestPrice { get; set; }
