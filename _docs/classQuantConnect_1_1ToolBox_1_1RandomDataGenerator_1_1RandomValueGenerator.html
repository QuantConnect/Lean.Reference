Generates a random decimal suitable as a price. This should observe minimum price variations if available in SymbolPropertiesDatabase, and if not, truncating to 2 decimal places.

Exceptions
ArgumentExceptionThrow when the referencePrice or maximumPercentDeviation is less than or equal to zero.
Parameters
securityTypeThe security type the price is being generated for
marketThe market of the security the price is being generated for
referencePriceThe reference price used as the mean of random price generation
maximumPercentDeviationThe maximum percent deviation. This value is in percent space, so a value of 1m is equal to 1%.
Returns
A new decimal suitable for usage as price within the specified deviation from the reference price
