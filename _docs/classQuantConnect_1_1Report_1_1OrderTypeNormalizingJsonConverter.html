Detailed Description

Normalizes the "Type" field to a value that will allow for successful deserialization in the OrderJsonConverter class.

All of these values should result in the same object:

[
{ "Type": "marketOnOpen", ... },
{ "Type": "MarketOnOpen", ... },
{ "Type": 4, ... },
]
