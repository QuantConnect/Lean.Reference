Detailed Description

Harami Cross candlestick pattern indicator

Must have:

  • first candle: long white (black) real body
  • second candle: doji totally engulfed by the first The meaning of "doji" and "long" is specified with SetCandleSettings The returned value is positive(+1) when bullish or negative(-1) when bearish; The user should consider that a harami cross is significant when it appears in a downtrend if bullish or in an uptrend when bearish, while this function does not consider the trend
